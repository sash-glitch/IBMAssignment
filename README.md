Web Scraping Project with BeautifulSoup
Overview

This project demonstrates how to extract data from web pages using Python and the BeautifulSoup library. It focuses on scraping financial data (e.g., Tesla and GME Quarterly Revenue tables) from HTML pages, cleaning the data, and storing it in pandas DataFrames for further analysis.

Features

Download HTML content from a webpage using requests.

Parse HTML using BeautifulSoup.

Extract tables by locating <table> and <tbody> elements.

Loop through rows and columns to retrieve specific data.

Store scraped data in structured pandas DataFrames.

Basic data cleaning to remove unwanted characters (like $ and ,) for numeric analysis.

Technologies Used

Python 3.x

BeautifulSoup4

Requests

Pandas
